# Lab 3 

## List of keys:
- `-h` `--help` - all commands
- `-l` `--list`
- `-v` `--version`
- `-r` `--recognize`
- `-o` `--oneline`
- `-n` `--number` - require an argument


## Run:
```
make
./terminal_command --list -r -o --version -v
./terminal_command -n 666
./terminal_command --number=555
make clean
```
